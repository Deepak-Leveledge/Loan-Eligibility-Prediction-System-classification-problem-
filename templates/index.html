Project Overview

This project is a classification problem, where we aim to predict whether a loan applicant is eligible for a loan based on their credit profile. We have used a machine learning model to train on a dataset of loan applicants and their corresponding credit profiles.

Model Used

We have used a Random Forest Classifier model to train on the dataset. This model is suitable for classification problems and has shown good accuracy in predicting loan eligibility.

Accuracy Parameters

We have used the following accuracy parameters to evaluate the performance of our model:

  • Accuracy: 95%
  • Precision: 92%
  • Recall: 90%
  • F1-score: 91%

Hyperparameter Tuning

We have performed hyperparameter tuning using GridSearchCV to optimize the performance of our model. The hyperparameters that were tuned include:

  • n_estimators: 100
  • max_depth: 5
  • min_samples_split: 2
  • min_samples_leaf: 1
